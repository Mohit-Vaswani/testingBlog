### Courses

- [Machine Learning Specialization](https://www.deeplearning.ai/courses/machine-learning-specialization/) by DeepLearning.AI is a comprehensive journey into the heart of ML. Led by industry expert Andrew Ng, offers an immersive exploration of key ML concepts, hands-on projects, and the skills needed to build intelligent systems. Dive into the world of ML, explore its practical applications, and pave the way for a career in this dynamic field. This specialization comprises three courses:
<a class="link" target="_blank" href="https://www.coursera.org/learn/machine-learning?specialization=machine-learning-introduction"> Supervised ML: Regression and Classification,</a>
<a class="link" target="_blank" href="https://www.coursera.org/learn/advanced-learning-algorithms?specialization=machine-learning-introduction"> Advanced Learning Algorithms</a> and
<a class="link" target="_blank" href="https://www.coursera.org/learn/unsupervised-learning-recommenders-reinforcement-learning?specialization=machine-learning-introduction"> Unsupervised Learning, Recommenders, Reinforcement Learning</a>.

- [Google Machine Learning Education](https://developers.google.com/machine-learning): Google provides a variety of resources, including <a class="link" target="_blank" href="https://developers.google.com/machine-learning/foundational-courses">foundational courses</a> that cover machine learning basics and core concepts. <a class="link" target="_blank" href="https://developers.google.com/machine-learning/guides">Guides</a> offer step-by-step walkthroughs for solving common machine learning problems using best practices, while <a class="link" target="_blank" href="https://developers.google.com/machine-learning/advanced-courses">advanced courses</a> teach tools and techniques for addressing a range of machine learning challenges.

### Guides

- [Machine Learning Mastery: Begin Your Journey Here](https://machinelearningmastery.com/start-here/): The common question that often comes my way is: "How do I start?" This all-encompassing guide not only addresses this question by offering insights into what to focus on and cultivating the right mindset but also delves into various subjects. It covers essentials such as mathematics for machine learning, Python programming, deep learning with Keras and PyTorch, natural language processing (NLP), computer vision, and more.

- [Machine Learning is Fun!](https://medium.com/@ageitgey/machine-learning-is-fun-80ea3ec3c471): Adam Geitgey's 8-part series covers a comprehensive range of machine learning topics, from basics to advanced concepts. The series includes parts on:
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-80ea3ec3c471">ML Fundamentals,</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-2-a26a10b68df3">Neural Networks,</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-3-deep-learning-and-convolutional-neural-networks-f40359318721">Deep Learning & Convolutional Neural Networks (CNNs),</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-4-modern-face-recognition-with-deep-learning-c3cffc121d78">Face Recognition,</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-5-language-translation-with-deep-learning-and-the-magic-of-sequences-2ace0acca0aa">Language Translation,</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-6-how-to-do-speech-recognition-with-deep-learning-28293c162f7a">Speech Recognition,</a>
<a class="link" target="_blank" href="https://medium.com/@ageitgey/abusing-generative-adversarial-networks-to-make-8-bit-pixel-art-e45d9b96cee7">Generative Adversarial Networks (GANs),</a> and
<a class="link" target="_blank" href="https://medium.com/@ageitgey/machine-learning-is-fun-part-8-how-to-intentionally-trick-neural-networks-b55da32b7196">Intentionally Tricking Neural Networks</a>. It offers a diverse and enriching learning experience for both beginners and advanced learners.

- [Machine Learning for Beginners](https://microsoft.github.io/ML-For-Beginners/#/): Microsoft's Azure Cloud Advocates introduce a comprehensive Machine Learning curriculum. Focused on classic machine learning with Scikit-learn, it excludes deep learning, covered in the <a class="link" target="_blank" href="https://microsoft.github.io/AI-For-Beginners/">'AI for Beginners'</a> course. Pair it with the <a class="link" target="_blank" href="https://microsoft.github.io/Data-Science-For-Beginners/#/">'Data Science for Beginners'</a> curriculum for a comprehensive learning experience.

- [Scikit-learn Docs](https://scikit-learn.org/stable/index.html) provides efficient tools for data mining and analysis, offering diverse ML algorithms for classification, regression, clustering, dimensionality reduction, model selection, and preprocessing tasks.

### Books

- [Hands on Machine learing](https://www.goodreads.com/en/book/show/32899495) by Aurélien Géron, you'll find a practical approach to machine learning. Dive into ML, grasp essential concepts, work on practical applications, and explore potential R&D directions, all while diving into real-world code examples.

- [Machine Learning with PyTorch and Scikit-Learn: Develop machine learning and deep learning models with Python](https://www.goodreads.com/book/show/60493099-machine-learning-with-pytorch-and-scikit-learn) is a comprehensive guide to using Python for data science, with a focus on scikit-learn for machine learning and PyTorch for deep learning. It covers topics like model evaluation, hyperparameter optimization, and deep learning, providing code examples for hands-on learning.

- [Pattern Recognition and Machine Learning](https://www.microsoft.com/en-us/research/uploads/prod/2006/01/Bishop-Pattern-Recognition-and-Machine-Learning-2006.pdf)  by Christopher M. Bishop is an insightful exploration of the core concepts in machine learning. This comprehensive guide delves into algorithms, offering a thorough understanding of pattern recognition and its applications in the evolving landscape of artificial intelligence.

- [An Introduction to Statistical Learning](https://www.statlearning.com/): As the extent and scale of gathering data expand across nearly every domain, a profound understanding of statistical learning has become indispensable. This book offers a comprehensive and accessible exploration of essential concepts in statistical learning, catering to individuals seeking to harness modern tools for data analysis. Widely recognized as the "Bible of Machine Learning," this book is suitable for a broad audience interested in leveraging contemporary methods for insightful data interpretation.

### Additional Reading

- [What is the history of artificial intelligence (AI)?](https://www.tableau.com/data-insights/ai/history): It provides a timeline of significant events in the history of AI, encompassing major advancements from the foundational efforts in the early 1900s to the significant leaps achieved in more recent times. For a deeper dive, consider exploring <a class="link" target="_blank" href="https://en.wikipedia.org/wiki/Timeline_of_artificial_intelligence">Timeline of artificial intelligence</a> at Wikipedia.

- [The Turing Test](https://plato.stanford.edu/entries/turing-test/): In 1950, Alan Turing revolutionized the field of AI with his paper "<a class="link" target="_blank" href="https://redirect.cs.umbc.edu/courses/471/papers/turing.pdf">Computing Machinery and Intelligence</a>," introducing the <a class="link" target="_blank" href="https://en.wikipedia.org/wiki/Turing_test">Turing Test</a> to evaluate machine intelligence through human-like interaction. This groundbreaking work profoundly influenced discussions on AI, establishing the foundation for future developments in the field. It sparked extensive discussions and debates in philosophy, cognitive science, and AI. For a brief overview, watch <a class="link" target="_blank" href="https://www.youtube.com/watch?v=3wLqsRLvV-c">The Turing test: Can a computer pass for a human?</a>

- [The Church-Turing Thesis](https://plato.stanford.edu/entries/church-turing/) posits that any effectively calculable function can be computed by a <a class="link" target="_blank" href="https://en.wikipedia.org/wiki/Turing_machine">Turing machine</a>. Proposed by Alonzo Church and Alan Turing, this foundational concept in computer science and mathematics underlies the universality of computation, asserting the equivalence of various models in expressing computational processes. For a brief overview & simple explanation, watch the video on <a class="link" target="_blank" href="https://www.youtube.com/watch?v=PLVCscCY4xI">Turing Machines</a>.
