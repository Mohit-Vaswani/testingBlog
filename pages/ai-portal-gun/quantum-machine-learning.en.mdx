With QML, you can explore transformative applications like optimizing drug discovery, enhancing encryption, solving complex optimization problems, and simulating quantum systems for scientific research.

### Courses

- [Quantum machine learning course](https://qiskit.org/learn/course/machine-learning-course) by IBM Quantum & Qiskit, covers quantum machine learning concepts, parameterized quantum circuits, training, and problem-solving. The course introduces recent developments in supervised and unsupervised learning, concluding with a showcase project.

- [University of TorontoX: Quantum Machine Learning](https://www.edx.org/learn/machine-learning/university-of-toronto-quantum-machine-learning): Emphasizes hands-on implementation using Python frameworks and features guest lectures from leading researchers. Objectives include understanding quantum states, contrasting quantum computing paradigms, implementing classical-quantum hybrid learning algorithms, and demonstrating coherent quantum machine learning protocols.

### Articles

- [Quantum ML and the Power of Data](https://blog.research.google/2021/06/quantum-machine-learning-and-power-of.html): Explores quantum computers' potential advantages in machine learning. It highlights quantum computers' ability to handle complex input correlations that challenge classical machines. The article discusses data-driven complexity and how classical models can rival quantum ones, concluding that classical methods can match quantum computers' power for certain problems.

- [Quantum machine learning: a classical perspective](https://royalsocietypublishing.org/doi/10.1098/rspa.2017.0551): Explores essential quantum computation, compares classical and quantum algorithm performance, considers problem perspectives in machine learning, and delves into quantum sampling methods.


### Papers

- [PennyLane: Automatic differentiation of hybrid quantum-classical computations](https://arxiv.org/abs/1811.04968v4) (2018): A Python framework, facilitates quantum and hybrid quantum-classical computation optimization and ML. It offers compatibility with qubit and continuous-variable models, enabling gradient calculations for variational quantum circuits, bridging classical and quantum techniques in optimization and machine learning. <a class="link" target="_blank" href="https://github.com/PennyLaneAI/pennylane">(code)</a> 

- [TensorFlow Quantum: A Software Framework for Quantum Machine Learning](https://arxiv.org/abs/2003.02989v2) (2020): An extension of TensorFlow, empowers quantum ML. It supports model construction, training, quantum circuit simulation, and combines classical and quantum computing within a single model, offering a high-level interface for hybrid quantum-classical models. <a class="link" target="_blank" href="https://github.com/tensorflow/quantum">(code)</a> 

- [A divide-and-conquer algorithm for quantum state preparation](https://arxiv.org/abs/2008.01511v2) (2020): Suggests a quantum state preparation method employing a divide-and-conquer algorithm to minimize the number of quantum gates needed. It involves recursively breaking down the target state into smaller sub-states that require fewer gates for preparation. <a class="link" target="_blank" href="https://github.com/qclib/qclib">(code)</a> 

- [Quantum Neuron: an elementary building block for machine learning on quantum computers](https://arxiv.org/abs/1711.11240v1) (2017): Introduces a quantum neuron as a fundamental element for ML. Utilizing a quantum circuit, it conducts nonlinear input data transformations, exhibiting promise in solving classification problems while addressing scalability challenges for larger models.

- [q-means: A quantum algorithm for unsupervised machine learning](https://arxiv.org/abs/1812.03584v2) (2019): Suggests "q-means," a quantum adaptation of the widely-used k-means clustering algorithm. It employs quantum phase estimation to determine eigenvalues from a distance matrix between data points. The study showcases q-means' potential in clustering and outlines challenges in its implementation on near-term quantum hardware.
